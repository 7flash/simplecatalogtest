--- Разрешение на редактирование поля продукта ---


В настройках поля схемы есть опция "Разрешить заполнение поля пользователем на странице создания/редактирования продукта (запрет может быть нужен для внутренних механизмов хранения данных сторонними плагинами)".

Кроме указанной информации в описании поля, эта опция может быть полезна если создать поле и в значение по-умолчанию установить скрипт отображения баннеров или некоторую подсказку, которую пользователь не сможет изменить при редактировании, а поле будет отображаться по всем остальным правилам.
